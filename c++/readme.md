#### Neural Network in C++

A small neural network project written in C++ for learning core deep learning concepts such as tensors, layers, forward propagation, backpropagation, and model structure.

This project is intentionally minimal and is meant for understanding implementation details rather than providing a production-ready framework.

#### Scope

Current goals:

- Build tensors and shape handling
- Implement layers and model structure
- Run forward and backward passes
- Keep the build simple with a Makefile
